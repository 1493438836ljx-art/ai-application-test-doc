### 2.6 核心方法

#### fetchSkillList
```javascript
const fetchSkillList = async () => {
  loading.value = true
  try {
    const params = {
      page: currentPage.value,
      size: pageSize.value,
      name: searchKeyword.value,
      category: categoryFilter.value,
      status: statusFilter.value,
    }
    const response = await getSkillList(params)
    if (response) {
      skillList.value = response.content || []
      total.value = response.totalElements || 0
    }
  } finally {
    loading.value = false
  }
}
```

#### handleCardClick
```javascript
const handleCardClick = async (item) => {
  loading.value = true
  const skillDetail = await getSkillDetail(item.id)
  detailSkill.value = skillDetail
  detailDialogVisible.value = true
  loading.value = false
}
```

#### handleSubmit
```javascript
const handleSubmit = async (formData, file) => {
  loading.value = true
  if (isEdit.value && currentSkill.value) {
    await updateSkill(currentSkill.value.id, formData, file)
    ElMessage.success('更新成功')
  } else {
    await createSkill(formData, file)
    ElMessage.success('创建成功')
  }
  fetchSkillList()
  loading.value = false
}
```

### 5.5 级联选择器选项构建

```javascript
const typeOptions = computed(() => {
  const options = []

  // 基本类型
  basicTypeOptions.forEach(t => options.push(t))

  // Array 复合类型
  const arrayChildren = [
    ...arrayElementTypeOptions,
    { value: 'File', label: 'File', children: fileTypeOptions }
  ]
  options.push({ value: 'Array', label: 'Array', children: arrayChildren })

  // File 复合类型
  options.push({ value: 'File', label: 'File', children: fileTypeOptions })

  return options
})
```

### 5.6 类型字符串解析

```javascript
const parseType = (typeStr) => {
  if (!typeStr) return { baseType: 'String', elementType: '', fileType: '' }

  // Array<File<SubType>>
  const arrayFileMatch = typeStr.match(/^Array<File<(.+)>>$/)
  if (arrayFileMatch) {
    return { baseType: 'Array', elementType: 'File', fileType: arrayFileMatch[1] }
  }

  // Array<Element>
  const arrayMatch = typeStr.match(/^Array<(.+)>$/)
  if (arrayMatch) {
    return { baseType: 'Array', elementType: arrayMatch[1], fileType: '' }
  }

  // File<SubType>
  const fileMatch = typeStr.match(/^File<(.+)>$/)
  if (fileMatch) {
    return { baseType: 'File', elementType: '', fileType: fileMatch[1] }
  }

  return { baseType: typeStr, elementType: '', fileType: '' }
}
```

### 5.7 核心方法

#### handleTypeChange
```javascript
const handleTypeChange = (index, val) => {
  let newType = 'String'

  if (val.length === 1) {
    newType = val[0]  // 基本类型
  } else if (val.length === 2) {
    if (val[0] === 'Array') {
      newType = `Array<${val[1]}>`
    } else if (val[0] === 'File') {
      newType = `File<${val[1]}>`
    }
  } else if (val.length === 3) {
    if (val[0] === 'Array' && val[1] === 'File') {
      newType = `Array<File<${val[2]}>>`
    }
  }

  updateParam(index, 'paramType', newType)
}
```

## 7. 交互流程

### 7.1 创建Skill流程

```
1. 用户点击"新建 Skill"按钮
2. 打开SkillFormDialog (isEdit=false)
3. 填写基本信息
4. 上传执行套件文件
5. 配置入参(可选)
6. 配置出参(可选)
7. 点击"确定"提交
8. 调用createSkill API
9. 刷新列表
```

### 7.2 编辑Skill流程

```
1. 用户点击卡片或下拉菜单"编辑"
2. 调用getSkillDetail获取完整数据
3. 打开SkillFormDialog (isEdit=true)
4. 表单自动填充现有数据
5. 修改字段/重新上传文件
6. 点击"确定"提交
7. 调用updateSkill API
8. 刷新列表
```

```
1. 用户点击卡片

3. 打开SkillDetailDialog
4. 展示完整信息(含参数表格)
5. 可下载执行套件
6. 可跳转编辑或发布
```
